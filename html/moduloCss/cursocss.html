Estructura de CSS

La estructura de CSS esta compuestas con selectores y propiedades, valores.

Los selectores son seleciones de los elementos,son como distintas formas de llamar a una funcion.
Estos seletores son varios y vamos nombrar algunos mas usados.
-Universales: sirve para selecionar todos los elementos de la paguina y se nombra con un (" * ").
-Por etiqueta: este selector funciona con solo porner las etiquetas de la estructura de html (" h2").. etc,.
-Clases: la clases son llamadas deforma de atributo y se la llama con un (".name").
En este curso usaremos el atributo "class" para el selector clase.
Propiedades:
Las propiedades son todo lo que le da estilo a una caja,texto, y imagen. Ahy miles de propiedades donde daremos de iniciar este curso para aprender lo suficiente para crear nuestra primera paguina wed.
